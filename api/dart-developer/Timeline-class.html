Static Methods

finishSync() → void

Finish the last synchronous operation that was started.

startSync(String name, {arguments}) → void

Start a synchronous operation labeled name. Optionally takes a Map of arguments. This operation must be finished before returning to the event queue.

timeSync(String name, dynamic function(), {arguments}) → dynamic

A utility method to time a synchronous function. Internally calls function bracketed by calls to startSync and finishSync.
