1. DomName('Element.insertAdjacentHTML')
void insertAdjacentHtml ( String where , String text , { NodeValidator validator , NodeTreeSanitizer treeSanitizer })

Parses text as an HTML fragment and inserts it into the DOM at the specified location.

The where parameter indicates where to insert the HTML fragment:

'beforeBegin': Immediately before this element. 'afterBegin': As the first child of this element. 'beforeEnd': As the last child of this element. 'afterEnd': Immediately after this element.

var html = '<div class="something">content</div>'; // Inserts as the first child document.body.insertAdjacentHtml('afterBegin', html); var createdElement = document.body.children 0 ; print(createdElement.classes 0 ); // Prints 'something'
