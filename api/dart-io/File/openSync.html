RandomAccessFile openSync ( { FileMode mode : FileMode.READ })

Synchronously open the file for random access operations. The result is a RandomAccessFile on which random access operations can be performed. Opened RandomAccessFile s must be closed using the RandomAccessFile.close method.

See open for information on the mode argument.

Throws a FileSystemException if the operation fails.
