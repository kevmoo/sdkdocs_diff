Uri requestedUri

The requested URI for the request.

The returend URI is reconstructed by using http-header fields, to access otherwise lost information, e.g. host and scheme.

To reconstruct the scheme, first 'X-Forwarded-Proto' is checked, and then falling back to server type.

To reconstruct the host, fisrt 'X-Forwarded-Host' is checked, then 'Host' and finally calling back to server.
