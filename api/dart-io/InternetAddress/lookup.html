Future < List < InternetAddress >> lookup ( String host , { InternetAddressType type : InternetAddressType.ANY })

Lookup a host, returning a Future of a list of InternetAddress s. If type is InternetAddressType.ANY , it will lookup both IP version 4 (IPv4) and IP version 6 (IPv6) addresses. If type is either InternetAddressType.IP_V4 or InternetAddressType.IP_V6 it will only lookup addresses of the specified type. The order of the list can, and most likely will, change over time.
