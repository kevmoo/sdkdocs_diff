An internet address.

This object holds an internet address. If this internet address is the result of a DNS lookup, the address also holds the hostname used to make the lookup. An Internet address combined with a port number represents an endpoint to which a socket can connect or a listening socket can bind.

Static Methods

lookup ( String host , { InternetAddressType type : InternetAddressType.ANY }) → Future < List < InternetAddress >>

Lookup a host, returning a Future of a list of InternetAddress s. If type is InternetAddressType.ANY , it will lookup both IP version 4 (IPv4) and IP version 6 (IPv6) addresses. If type is either InternetAddressType.IP_V4 or InternetAddressType.IP_V6 it will only lookup addresses of the specified type. The order of the list can, and most likely will, change over time.
